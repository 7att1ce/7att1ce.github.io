第1章 绪论

1.1 信息的概念

  1. 信息、情报、知识、消息及信号间的区别与联系
    • 情报是人们对于某个特定对象所见、所闻、所理解而产生的知识
    • 知识是人们根据某种目的，从自然界收集得来的数据中，整理、概括、提取得到有价值的、人们所需的信息
      • 知识是一种具有普遍和概括性质的高层次的信息
    • 用文字、符号、数据、语言、音符、图片等能够被人们感觉器官所感知的形式，把客观物质运动和主观思维活动的状态表达出来就成为消息
      • 消息是信息的载体，同一则信息可用不同的消息形式来载荷
    • 把消息变换成适合信道传输的物理量，这种物理量称为信号
      • 信号携带着消息，它是消息的运载工具
  2. 香农信息的定义
    • 定义：信息是事物运动状态或存在方式的不确定性的描述
    • 消息传递过程的特点
      • 收信者在收到消息以前不知道消息的具体内容
      • 收信者无法判断是描述这种状态还是那种状态
      • 收信者不能断定所得到的信息是否正确和可靠
    • 在通信系统中形式上传输的是消息，但实质上传输的是信息
    • 通信的结果是消除或部分消除不确定性从而获得信息
    • 不确定性包括：随机性、粗糙性、模糊性
  3. 香农信息的度量
    • 消除多少不确定性，就获得多少信息量
    1. 样本空间：所有可能选择的消息的集合
    2. 概率测度：对于离散消息的集合，概率测度就是对每一个可能选择的消息指定一个概率(非负的，且总和为1)
    3. 概率空间：一个样本空间和它的概率测度，数学模型为：

      其中，就是选择符号作为消息的概率，成为先验概率，把条件概率称为后验概率
    4. 香农信息定义的局限性
      • 这个定义的出发点是假定事物状态可以用一个以经典集合论为基础的概率模型来描述。然而对实际某些事物运动状态或存在状态要寻找一个合适的概率模型往往是非常困难的。对于某些情况来讲，是否存在这样一种模型还值得探讨。
      • 这个定义和度量没有考虑收信者的主观特征和主观意义，也撇开了信息的具体含义、具体用途、重要程度和引起后果等因素
  4. 信息的广义概念
    • 语法信息：是事物运动状态和状态改变的方式的本身
    • 语义信息：是事物运动状态和方式的具体含义
    • 语用信息：是事物运动状态和方式及其含义对观察者的效用，或者是相对于某种目的的效用

1.2 信息论研究的对象、目的和内容

  1. 信息论研究的对象
    1. 信息源：信源是产生消息和消息序列的源
    2. 编码器：编码是把消息变换成信号的措施，而译码就是编码的反变换。编码器输出的是适合信道传输的信号，信号携带者信息，它是消息的载荷者
    3. 信道：信道是指通信系统把载荷消息的信号从甲地传输到乙地的媒介或通道
    4. 译码器：译码就是把信道输出的编码信号进行反变换，一般认为这种变换是可逆的
    5. 信宿：信宿是消息传送的对象，即收消息的人或机器
  2. 信息论研究的目的：要找到信息传输过程的共同规律，以提高信息传输的可靠性、有效性、保密性和认证性
  3. 信息论研究的内容：狭义信息论、一般信息论、广义信息论

第2章 离散信源及其信息测度

2.1 信源的数学模型及分类

  1. 信源输出的消息用随机变量描述
    • 单符号离散信源：信源输出的消息数是有限的或可数的，而且每次只输出其中一个消息
    • 可用一维离散型随机变量X来描述这些信源的输出，这样的信源称为离散信源，它的数学模型就是离散型的概率空间
    • 可用一维的连续型随机变量X来描述信息，则这种信源成为连续信源，其数学模型是连续的概率空间

2.2 离散信源的信息熵及其性质

一、自信息

  • 自信息->信源中每个消息的出现会携带的信息量
  • 信息熵->整个信源能输出的信息量
    • 信息的度量(信息量)和不确定性消除的程度有关,消 除的不确定性=获得的信息量
    • 不确定性指随机性,可用概率论和随机过程测度
  • 定义: 设离散信源X的概率空间为

    事件的自信息量：发生所含有的信息量
  • 代表两种含义
    • 当事件发生以前,表示事件发生的不确定性
    • 当事件发生以后,表示事件所含有(或所提供) 的信息量
  • 自信息量的单位取决于对数的底
    • 底为2，单位为 “比特(bit, binary unit)”
    • 底为e，单位为 “奈特(nat, nature unit)”
    • 底为10，单位为 “哈特(hat, Hartley)”
  • 一般采用以“2”为底的对数,常把底数“2”略去不写
